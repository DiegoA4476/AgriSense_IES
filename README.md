# AgriSense

![CI](https://github.com/detiuaveiro/ies2526-group-project-ies2526_g206/actions/workflows/ci.yml/badge.svg)

## Abstract
 A livestock monitoring platform that processes body temperature, movement, and location sensor data 
to detect health anomalies in farm animals.

## Personas
### Manager
Responsible for registering new farms.

---

| Use Cases| User Stories |
|-|-|
| Register a farmer | <u>As a</u> **Manager**, <u>I want to</u> register a farmer <u>so that</u> new farmers can access the system and manage their farms. |
| Delete a farmer | <u>As a</u> **Manager**, <u>I want to</u> delete a farmer's account <u>so that</u> inactive or unauthorized users no longer have access to the system. |
| Register a farm | <u>As a</u> **Manager**, <u>I want to</u> register a farm <u>so that</u> it can be officially managed and monitored within the system. |
| Delete a farm |<u>As a</u> **Manager**, <u>I want to</u> delete a farm <u>so that</u> outdated or closed farms are no longer stored in the system. |

### Farmer
Responsible for managing the barns and monitoring the animal's vital signs.  

---

| Use Cases| User Stories |
|-|-|
| Add a barn | <u>As a</u> **Farmer**, <u>I want to</u> add a barn to my farm <u>so that</u> I can accurately represent my farm in the system. |
| Remove a barn | <u>As a</u> **Farmer**, <u>I want to</u> remove a barn from my farm <u>so that</u> I can keep my farm structure accurate and up to date when a barn is no longer in use. |
| Add an animal to a barn | <u>As a</u> **Farmer**, <u>I want to</u> add an animal to a barn <u>so that</u> so that I can keep my barn up to date when a new animal arrives. |
| Remove an animal from a barn | <u>As a</u> **Farmer**, <u>I want to</u> remove an animal from a barn <u>so that</u> I can keep my barn up to date when an animal is no longer alive. |
| View a barn's dashboard | <u>As a</u> **Farmer**, <u>I want to</u> view a barn's dashboard <u>so that</u> so that I can keep up with the current status of the barn. |
| View a barn's animal list | <u>As a</u> **Farmer** <u>I want to</u> view a barn's animal list <u>so that I</u> see what animals are in that barn. |
| View an animal's dashboard | <u>As a</u> **Farmer**, <u> I want to</u> view an animal's dashboard <u>so that</u> I can see its vital signs. |
| View critical notifications |<u>As a</u> **Farmer**, <u>I want to</u> view critical notifications when an animal exhibits abnormal signs <u>so that</u> I can respond quickly and avoid a more serious situation. |
