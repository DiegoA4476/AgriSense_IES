# ies2526-group-project-ies2526_g206
## Abstract
    This project presents a livestock monitoring platform designed to supervise animal health and behavior across multiple barns within a farm environment. The system integrates a data acquisition layer where each barn, housing a distinct animal species, continuously collects individual animal parameters including body temperature, movement, heart rate, respiratory rate, location, and posture (standing or resting). In addition, environmental conditions within each barn are monitored to provide contextual information that may influence animal wellbeing.
